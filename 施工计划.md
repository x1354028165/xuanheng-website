- [ ] **P0-5** 基础布局组件

  - `frontend/` + `cms/` 均有 `.gitignore`
  - 初始 commit 推送到 main

- [ ] **P0-8** 安装配置 shadcn/ui
  - `npx shadcn-ui@latest init`
  - 配置主题色：--primary: #1A3FAD，--accent: #38C4E8

- [ ] **P0-9** 配置 Ant Design（CMS插件目录）
  - `cd cms && npm install antd`

- [ ] **P0-X** DeepSeek 翻译插件（Strapi 服务端）
  - 创建 `cms/src/api/translate/` 自定义 API

  - Strapi Admin Extension 注入「一键翻译」按钮

  - 测试：输入中文 → 返回英文，专业术语正确

- [ ] **P0-Y** next-intl 全量配置（8语言底座，3语言激活）
  - `i18n.config.ts` 定义全部8个locale
  - `middleware.ts` 只对 zh-CN/en-US/zh-TW 放行，其余重定向至 en-US
  - zh-TW 路由接入 OpenCC（`opencc-js`）处理
  - `messages/zh-CN.json` 建立完整key结构（Phase 2 填充翻译）
