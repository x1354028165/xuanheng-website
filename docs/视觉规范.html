零、设计哲学 Design Philosophy

旭衡电子官网设计哲学：借鉴阳光电源(Sungrow)、Tesla等能源行业头部企业的极简大气风格。

核心原则

原则说明
① 克制用色页面90%以白/浅灰/深色为基底，品牌青蓝 #38C4E8 仅用于CTA按钮、关键链接、数字统计强调，不做大面积背景色
② 图片优先用高质量能源场景图（光伏电站、储能系统、智慧能源）传递品牌价值，而非靠颜色
③ 留白即设计大 margin / padding，不填满，给用户呼吸感
④ 一个强调色全站只有 #38C4E8 一个强调色，辅色 #1A3FAD 只用于深色区块背景，不做按钮色
⑤ 字体做层次靠字号、字重、颜色深浅区分信息层级，不靠色块堆砌

七、动效规范 Motion

动效应克制、有目的——引导注意力，而非分散注意力。遵循 Stripe/Linear 简洁动效哲学。

transition-fast
150ms ease-out
hover 状态切换、颜色变化
transition-base
200ms ease-out
按钮/链接交互、阴影变化
transition-slow
300ms ease-in-out
面板展开/收起、抽屉动画
transition-page
400ms ease-in-out
页面切换、路由过渡
// CSS Custom Properties --transition-fast: 150ms ease-out; --transition-base: 200ms ease-out; --transition-slow: 300ms ease-in-out; --transition-page: 400ms ease-in-out; // Usage .btn { transition: all var(--transition-base); } .card { transition: all var(--transition-base); } .panel { transition: all var(--transition-slow); }

动效原则

原则说明
有目的性每个动效必须引导用户注意力或提供反馈，禁止纯装饰动画
保持克制使用 ease-out 让元素快速到位，避免弹性（bounce）效果
可访问性尊重 prefers-reduced-motion，提供关闭动效选项
性能优先仅动画 transform 和 opacity，避免动画 width/height/margin
