七、动效规范 Motion

动效应克制、有目的——引导注意力，而非分散注意力。遵循 Stripe/Linear 简洁动效哲学。

transition-fast
150ms ease-out
hover 状态切换、颜色变化
transition-base
200ms ease-out
按钮/链接交互、阴影变化
transition-slow
300ms ease-in-out
面板展开/收起、抽屉动画
transition-page
400ms ease-in-out
页面切换、路由过渡
// CSS Custom Properties --transition-fast: 150ms ease-out; --transition-base: 200ms ease-out; --transition-slow: 300ms ease-in-out; --transition-page: 400ms ease-in-out; // Usage .btn { transition: all var(--transition-base); } .card { transition: all var(--transition-base); } .panel { transition: all var(--transition-slow); }

动效原则

原则说明
有目的性每个动效必须引导用户注意力或提供反馈，禁止纯装饰动画
保持克制使用 ease-out 让元素快速到位，避免弹性（bounce）效果
可访问性尊重 prefers-reduced-motion，提供关闭动效选项
性能优先仅动画 transform 和 opacity，避免动画 width/height/margin

十二、图标规范 Icons

本项目使用专业定制设计图标（由设计师提供），非系统符号或开源图标库。

使用规范

项目规范
格式SVG（支持任意尺寸缩放，保持清晰度）
尺寸规格16px / 20px / 24px / 32px / 48px
线宽1.5px（小尺寸） / 2px（标准尺寸）
风格线性图标为主，与品牌青蓝主色 #38C4E8 搭配使用
颜色默认继承父元素 currentColor，便于深色/浅色背景复用
交付格式SVG文件 + React组件（TSX）双版本
存放路径frontend/src/components/icons/
命名规范PascalCase，如 IconSolar.tsx / IconBattery.tsx
⚠️ 禁止使用 Emoji 作为功能性图标；临时占位可用，上线前必须替换为定制图标。
