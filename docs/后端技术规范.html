1.2 Strapi 在整体架构中的角色

  • 唯一内容数据源（Single Source of Truth）：所有页面内容、产品信息、解决方案、新闻等均通过 Strapi 管理
  • REST API 提供者：供 Next.js 在构建时（SSG）和增量静态再生（ISR）时拉取数据
  • 内置媒体库：对接阿里云 OSS，提供统一的文件上传和管理
  • Webhook 触发器：内容发布/更新时自动触发 Next.js ISR 刷新
  • i18n 多语言支持：内置国际化插件，支持 zh-CN / en-US 等多语言内容管理

二、Strapi 内容类型设计

2.1 设计原则

  • 所有面向前台展示的内容必须开启 Strapi i18n 插件（多语言字段）
  • 富文本字段使用 Strapi 内置 Rich Text（基于 Slate.js），不引入第三方编辑器
  • 关系字段（如产品 ↔ 解决方案）使用 Strapi 多对多关系（many-to-many）
  • 所有 Collection Type 默认包含字段：createdAt / updatedAt / publishedAt / locale
  • UID 字段（slug）统一使用英文，基于 name 自动生成，确保 URL 友好
  • 排序字段 sortOrder 默认值为 0，数值越小越靠前

三、API 设计规范

3.1 基础规则

  • 全部使用 Strapi 原生 REST API，不自定义路由（除非必要）
  • 基础路径：https://api.alwayscontrol.com.cn/api/
  • 认证：Bearer Token
    • 只读 Token：给 Next.js 使用，仅 find / findOne 权限
    • 管理 Token：仅 Strapi 后台内部使用，不对外暴露
  • 分页：默认 pagination[pageSize]=25，最大 100
  • 排序：sort[0]=sortOrder:asc
  • 多语言：locale=zh-CN（所有请求必须指定 locale）
  • 关系填充：populate=*（浅层）或 populate[relation]=*（指定关系）

解决编辑手动复制粘贴翻译效率衰减问题，在 Strapi 后台实现「一键翻译」按钮，服务端调用 DeepSeek API（不受国内网络限制）。

项目说明
实现方式Strapi 自定义 API Route：/api/translate
输入格式{ text: string, from: 'zh-CN', to: 'en-US' | 'de' | 'fr' | 'pt' | 'es' | 'ru' }
调用接口https://api.deepseek.com/chat/completions，model: deepseek-chat

系统 Prompt：

你是专业翻译，专注于能源管理/工业物联网领域术语。将以下${from}文本翻译为${to}，保持专业术语准确，保留Markdown格式。

后端 Controller 实现（cms/src/api/translate/controllers/translate.ts）：

// cms/src/api/translate/controllers/translate.ts
export default {
  async translate(ctx) {
    const { text, from, to } = ctx.request.body
    const response = await fetch('https://api.deepseek.com/chat/completions', {
      method: 'POST',
      headers: {
        'Authorization': `Bearer ${process.env.DEEPSEEK_API_KEY}`,
        'Content-Type': 'application/json'
      },
      body: JSON.stringify({
        model: 'deepseek-chat',
        messages: [
          {
            role: 'system',
            content: `你是专业翻译，专注于能源管理/工业物联网领域。将${from}翻译为${to}，保持术语准确，保留Markdown格式。`
          },
          { role: 'user', content: text }
        ]
      })
    })
    const data = await response.json()
    ctx.body = { translation: data.choices[0].message.content }
  }
}

4.3 多语言扩展模式

Strapi i18n 插件的工作原理：

  • 每个多语言记录在同一张表中存储，通过 locale 字段区分语言版本
  • 不同语言版本通过 {content_type}_localizations_links 中间表建立关联
  • zh-CN 为主语言（默认 locale），en-US 为翻译语言
  • Media 字段（图片/文件）默认在语言版本间共享，不重复上传
  • 非 i18n 字段（如 slug、sortOrder）在各语言版本中保持一致
-- 示例：同一产品的中英文版本在 products 表中各占一行
-- id=1: name="能源管理系统 EMS Pro", locale="zh-CN"
-- id=2: name="Energy Management System EMS Pro", locale="en-US"
-- 通过 products_localizations_links 关联：
--   (product_id=1, inv_product_id=2)
--   (product_id=2, inv_product_id=1)

5.3 Next.js revalidate 端点实现

// app/api/revalidate/route.ts
import { revalidatePath } from 'next/cache';
import { NextRequest, NextResponse } from 'next/server';

export async function POST(req: NextRequest) {
  // 1. 验证 Secret Token
  const secret = req.headers.get('authorization')?.replace('Bearer ', '');
  if (secret !== process.env.REVALIDATE_SECRET) {
    return NextResponse.json({ error: 'Unauthorized' }, { status: 401 });
  }

  // 2. 解析 Strapi Webhook payload
  const body = await req.json();
  const model = body.model; // 'product', 'solution', 'article' 等

  // 3. 按内容类型映射需刷新的路径
  const pathMap: Record<string, string[]> = {
    'product':           ['/zh-CN/products', '/en-US/products'],
    'solution':          ['/zh-CN/solutions', '/en-US/solutions'],
    'article':           ['/zh-CN/about/news', '/en-US/about/news'],
    'compatible-brand':  ['/zh-CN', '/en-US', '/zh-CN/ecosystem', '/en-US/ecosystem'],
    'page-content':      ['/zh-CN', '/en-US'],
    'seo-config':        ['/zh-CN', '/en-US'],
    'faq':               ['/zh-CN/support', '/en-US/support'],
    'case-study':        ['/zh-CN/cases', '/en-US/cases'],
    'job-opening':       ['/zh-CN/about', '/en-US/about'],
  };

  const paths = pathMap[model] || ['/zh-CN', '/en-US'];

  // 4. 刷新所有受影响路径
  for (const path of paths) {
    revalidatePath(path, 'page');
  }

  console.log(`[Revalidate] model=${model}, paths=${JSON.stringify(paths)}`);
  return NextResponse.json({ revalidated: true, model, paths });
}

5.4 安全验签

  • Secret Token 方式（推荐）：Strapi Webhook Header 携带 Authorization: Bearer {SECRET}，Next.js 端对比环境变量 REVALIDATE_SECRET
  • Token 要求：至少 32 位随机字符串，使用 openssl rand -hex 32 生成
  • 防重放（可选）：在 Webhook payload 中加入 timestamp 字段，Next.js 端拒绝超过 5 分钟的请求
  • IP 白名单（可选）：在 Vercel 或 Nginx 层限制仅允许 Strapi 服务器 IP 访问 /api/revalidate

7.2 Next.js API Route（联系表单 + 企微通知）

// app/api/contact/route.ts
import { NextRequest, NextResponse } from 'next/server';

const STRAPI_URL = process.env.STRAPI_URL!;
const STRAPI_TOKEN = process.env.STRAPI_ADMIN_TOKEN!;

export async function POST(req: NextRequest) {
  const body = await req.json();
  const { name, email, company, country, message, subject, turnstileToken, _honey } = body;

  // 1. Honeypot 检查（隐藏字段应为空）
  if (_honey) {
    return NextResponse.json({ success: true }); // 假装成功，不处理
  }

  // 2. Turnstile 验证
  const turnstileRes = await fetch('https://challenges.cloudflare.com/turnstile/v0/siteverify', {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({
      secret: process.env.TURNSTILE_SECRET_KEY,
      response: turnstileToken,
    }),
  });
  const turnstileData = await turnstileRes.json();
  if (!turnstileData.success) {
    return NextResponse.json({ error: '验证失败，请重试' }, { status: 400 });
  }

  // 3. 基础验证
  if (!name || !email || !message) {
    return NextResponse.json({ error: '缺少必填字段' }, { status: 400 });
  }
  if (!/^[^\s@]+@[^\s@]+\.[^\s@]+$/.test(email)) {
    return NextResponse.json({ error: '邮箱格式不正确' }, { status: 400 });
  }

  // 4. 写入 Strapi
  const strapiRes = await fetch(`${STRAPI_URL}/api/leads`, {
    method: 'POST',
    headers: {
      'Content-Type': 'application/json',
      'Authorization': `Bearer ${STRAPI_TOKEN}`,
    },
    body: JSON.stringify({
      data: {
        name, email, company, country, message, subject,
        source: req.headers.get('referer') || '',
        status: 'new',
      },
    }),
  });

  if (!strapiRes.ok) {
    console.error('[Contact] Strapi write failed:', await strapiRes.text());
    return NextResponse.json({ error: '提交失败，请稍后再试' }, { status: 500 });
  }

  // 5. 企微 Webhook 通知（异步，不阻塞响应）
  fetch(process.env.WECOM_WEBHOOK_URL!, {
    method: 'POST',
    headers: { 'Content-Type': 'application/json' },
    body: JSON.stringify({
      msgtype: 'markdown',
      markdown: {
        content: [
          '## 🔔 新客户线索',
          `**姓名**：${name}`,
          `**邮箱**：${email}`,
          company ? `**公司**：${company}` : '',
          country ? `**国家**：${country}` : '',
          subject ? `**主题**：${subject}` : '',
          `**留言**：${message.substring(0, 200)}${message.length > 200 ? '...' : ''}`,
          `**来源**：${req.headers.get('referer') || '未知'}`,
        ].filter(Boolean).join('\n'),
      },
    }),
  }).catch(err => console.error('[WeCom] Webhook failed:', err));

  return NextResponse.json({ success: true });
}

完全自研最灵活，无外部依赖工期 3–6 个月，当前阶段过重❌ 不推荐
SaaS CMS（Contentful / Sanity）上手快多语言字典管理等自定义逻辑受限，长期按量计费❌ 不推荐

决策关键点： 确认后端团队 Node.js 熟练程度。Strapi 适合快速上线，Directus 适合数据模型复杂场景。

11.1b CMS 自定义 UI 组件库：Ant Design

Strapi CMS 自定义插件 / 扩展层使用 Ant Design（antd v5） 作为 UI 组件库。

使用场景：

  • Strapi Admin 自定义页面（如仪表盘 CMS-01、账号管理 CMS-22）
  • 自定义字段组件（如产品参数表编辑器、兼容矩阵编辑器）
  • 仪表盘 Widget（数据统计卡片、图表面板）

安装方式（在 CMS 插件目录下）：

cd cms && npm install antd

使用原则：

  • 仅在 cms/src/admin/ 目录下使用 Ant Design，不影响前台 Next.js 项目
  • 优先使用 antd 提供的组件：Table / Form / Modal / Tabs / Statistic / Card 等
  • 主题色通过 ConfigProvider 配置为品牌色 #1A3FAD

11.4 文件下载安全

需求（来自 PRD §8）：

  • 权限受控文件不能直接暴露 CDN 链接
  • 防止链接外泄后被无限次使用

推荐方案： 对象存储签名临时链接（Pre-signed URL），有效期 5 分钟

  • 阿里云 OSS / 腾讯云 COS 均支持此功能
  • 用户点击下载时，后端实时生成签名链接并返回，不存储静态链接
