EP-04 开发者中心（Tooltip 提示，无独立页面）
📌 初版不建独立页面。导航入口和 Hero CTA 保留，点击/悬停后显示 Tooltip，引导用户前往联系我们留资。完整功能（API 文档 / 权限控制 / SDK）后续迭代启动。

交互规格

触发方式： - 导航栏「开发者中心」按钮：悬停（PC）或点击（移动端） - 首页 Hero「开发者中心」CTA 按钮：点击 Tooltip 内容： ┌──────────────────────────────┐ │ 🚀 开发者中心即将上线 │ │ API 文档 / SDK 正在准备中 │ │ [立即留资，优先获取通知 →] │ └──────────────────────────────┘ 点击 [立即留资]： 跳转联系我们页面（/zh-CN/contact） 意向分类自动预填「API 接口接入需求」 样式要求： 背景深色（#1a3a5c），白色文字 圆角卡片，带小三角指向触发元素 PC：悬停 300ms 后出现，鼠标移出后消失 移动端：点击出现，点击页面其他区域消失
