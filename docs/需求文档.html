说明

本文档为功能需求清单，列出官网所有页面及 CMS 模块的详细功能规格与验收标准，供开发团队确认范围使用。每个页面/模块按以下格式描述：区块结构（布局图）、功能清单表格、验收标准（绿色框）。

四、全局功能需求

EP-09 多语言
功能点说明
zh-CN / en-US 双语全量覆盖核心维护语言
zh-TW 自动繁简转换由 zh-CN 自动生成
Header 语言切换器仅显示已发布语言
URL 多语言结构（/zh-CN/... /en-US/...）SEO 独立索引
de / fr / pt / es / ru 机器翻译辅助以 en-US 为源，机器翻译 + 人工校对
内容回退机制未翻译内容按链回退，不显示空白
切换语言后 URL 变更，页面内容正确切换，SEO 元数据同步更新。
EP-10 全局搜索
功能点说明
Header 常驻搜索框跨 FAQ / 兼容品牌 / 产品 / 文档全站索引
跨全站索引范围FAQ / API 文档段落 / 兼容品牌 / 固件版本 / 产品页
CMS 内容发布后自动更新索引内容发布触发索引刷新，无需手动触发
EP-11 表单通知
功能点说明
企业微信群机器人通知（按意向分类路由）Webhook URL 在 CMS 通知配置后台维护
内部邮件通知（按意向分类路由至指定邮箱）收件邮箱在 CMS 后台配置
用户侧确认回执邮件邮件模板在 CMS 后台编辑，支持多语言
表单提交 5 分钟内，企微群和指定邮箱均收到通知；用户邮箱收到确认邮件。
EP-12 SEO 基础
功能点说明
SSR（服务端渲染）强制不使用纯 SPA，保证搜索引擎可索引
每页 title / description / OG Tags 可配置在 CMS SEO 配置模块独立维护
Sitemap 自动生成内容发布后自动更新
robots.txt 配置Staging 全站 noindex
EP-13 隐私合规
功能点说明
Cookie 授权弹窗（GDPR/LGPD）首次访问弹出，支持多语言
隐私政策页（多语言）⚠️ 法务审核后上线，不使用模板占位
EP-14 SSO / 访问控制
📌 初版上线时所有内容公开，无登录入口。SSO 及权限体系后续迭代接入。
功能点说明
初版全内容公开无登录入口，所有页面匿名可访问
SSO 接入（后续迭代）统一账号系统，Header 显示登录按钮
文件下载权限分层（后续迭代）无权限时引导联系商务
